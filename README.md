# Paraitroopers

A web-based arcade shooter game where you control a ground-to-air cannon defending against paratroopers.

## Controls

- **A / D** - Rotate cannon left/right
- **Space** - Fire projectile
- **Q** - Fire special ammo (press again to arm, press again to detonate)
- **ESC** - Pause/Resume
- **R** - Restart (when game over)

## Gameplay

- Airplanes fly across the sky dropping paratroopers
- Shoot paratroopers before they reach the ground
- If 5 paratroopers land safely from the same side, they plant explosives and it's game over
- Special ammo creates an explosion with area damage

## Run

Open `index.html` in any modern web browser.
